What is CSS?

  • CSS stands for Cascading Style Sheet
  • Cascading Style Sheets (CSS) is used to format the layout of a webpage.

With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!

Using CSS

CSS can be added to HTML documents in 3 ways:

  1. Inline
  2. Internal
  3. External

Inline CSS

An inline CSS is used to apply a unique style to a single HTML element.

An inline CSS uses the style attribute of an HTML element.

Basic Example:

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1 style="color:blue;" >My First Heading</h1>
        <p style="color:red;" >My first paragraph.</p>
    </body>
</html>

Internal CSS

An internal CSS is used to define a style for a single HTML page.

An internal CSS is defined in the <head> section of an HTML page, within a <style> element.

Achieved with the use of selectors(defines single or multiple element of html)

Basic Example:

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
        <style>
            body {background-color: powderblue;}
            h1   {color: blue;}
            p    {color: red;}
        </style>
    </head>
    <body>
        <h1>My First Heading</h1>
        <p>My first paragraph.</p>
    </body>
</html>

External CSS

An external style sheet is used to define the style for many HTML pages

To use an external style sheet, add a link to it in the <head> section of each HTML page:

Achieved with the use of selectors(defines single or multiple element of html)

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
        <link rel = "stylesheet" href = "path of css">
    </head>
    <body>
    </body>
</html>

CSS Colors

We can use colors in three types:

  1. Using Color Name like red, blue, white, etc
  2. Using Hex Code like #fff
  3. using rgb like rgb(f,f,f)

Text Color Example

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1 style="color:blue;">My First Heading</h1>
        <p style="color:red;">My first paragraph.</p>
    </body>
</html>

background Color Example

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body style="background-color:black;">
        <h1 style="color:white;">My First Heading</h1>
        <p style="color:white;">My first paragraph.</p>
    </body>
</html>

CSS Border Color

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body style="background-color:black;">
        <h1 style="color:white; border ="1px solid white">My First Heading</h1>
    </body>
</html>

Text Alignment

The text-align property is used to set the horizontal alignment of a text.

A text can be left or right aligned, centered, or justified.

Basic Example

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <p style = "text-align: center;" > Centered aligned Text </p>
        <p style = "text-align: left;" > Left aligned Text </p>
        <p style = "text-align: right;" > Right aligned Text </p>
    </body>
</html>

Font Family

The font-family property is used to set the font style of a text.

Basic Example

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <p style = "font-family: monospace;" > This is monospace font </p>
        <p> This is default font </p>
    </body>
</html>

Font Size

The font-size property is used to set the font size of a text.

Basic Example

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <p style = "font-size: 30px;" > Large Text </p>
        <p> This is Normal Text </p>
    </body>
</html>

Text Decoration

The text-decoration property is used to set the underline for a text.

Value can be any style like dotted, double, solid,...etc

Basic Example

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <p style = "text-decoration: underline #f00 solid; font-size:25px;" > Underlined Text </p>
    </body>
</html>

CSS Padding

The CSS padding properties are used to generate space around an element's content, inside of any defined borders.

All the padding properties can have the following values:

  • length - specifies a padding in px, pt, cm, etc.
  • Percent - specifies a padding in % of the width of the containing element
  • inherit - specifies that the padding should be inherited from the parent element

Note: Negative values are not allowed.

    <!DOCTYPE html>
    <html>
        <head>
            <title>Page Title</title>
        </head>
        <body>
            <p style="padding: 10px; border:1px solid black;">Hello World</p>
            <p style="padding-left: 50px; border:1px solid black;">Padding from left</p>
        </body>
    </html>

CSS Margin

The CSS margin properties are used to create space around elements, outside of any defined borders.

All the margin properties can have the following values:

  • auto - the browser calculates the margin
  • length - specifies a margin in px, pt, cm, etc.
  • Percent - specifies a margin in % of the width of the containing element
  • inherit - specifies that the margin should be inherited from the parent element

Note: Negative values are allowed.

    <!DOCTYPE html>
    <html>
        <head>
            <title>Page Title</title>
        </head>
        <body>
            <p style="margin: 10px; border:1px solid black;">Hello World</p>
            <p style="margin-top: 50px; border:1px solid black;">Margin from top</p>
        </body>
    </html>
