What is HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content

Features of HTML

  • It is platform-independent.
  • Images, videos, and audio can be added to a web page.
  • Hypertext can be added to the text.
  • It is a markup language.
  • It is interpreted language
  • It can be integrated with other languages like CSS, JavaScript etc.
  • Semantic Structure.
  • local Storage & Indexed DB â€“ Client-side data storage.
  • Offline Capabilities (PWA) with Cache API & Service Workers.

History of HTML

  • In 1989, Tim Berners Lee established www and created HTML in 1991. 
  • From 1995 to 1997, he started working on versions of HTML.
  • In 1999, a committee was organized that made HTML4.0 a standard.
  • HTML4.0 is used by many, even today. However, the stable version of HTML is 5.0 also known as HTML5.

    </head>
    <body>
        <h1>My First Heading</h1>

HTML Lists

Our day-to-day lives revolve around lists. For example, We purchase items while shopping. A list of all our items is included on the bill we receive from the shopkeeper. Similarly, web developers use lists to display the data.

HTML lists are used to display the data in an ordered and unordered form. List contains one or more list elements.

The HTML list is of three types:

  • An unordered list.
  • An ordered list.
  • A definition list.

HTML Unordered List

  • Unordered lists have no sequence.
  • They are more like bullet points. 
  • They are created by using <ul> tag.
  • <ul> tag has its corresponding </ul> tag.
  • For items, we use <li> tag also known as the List Item tag.
<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <p>Fruits List.</p>
        <ul type = "square">
            <li>Apple</li>
            <li>Mango</li>
            <li>Banana</li>
        </ul>
    </body>
</html>

type attribute:

Type attribute is used to specify the type of bullet.

There are three types of bullets:

  • disc
  • square
  • circle

HTML Ordered List

  • Ordered lists have a sequence.
  • They are a more like numbered list.
  • They are created by using the <ol> tag.
  • For items, we use <li> tag also known as the List Item tag.
<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <p>Fruits List.</p>
        <ol type = "a">
            <li>Apple</li>
            <li>Mango</li>
            <li>Banana</li>
        </ol>
    </body>
</html>

type attribute:

Type attribute is used to specify the type of numbering we want.

There are five types of numbering:

  1. Uppercase Roman Numerals (I,II,III,...)
  2. Lowercase Roman Numerals (i,ii,iii,...)
  3. Numerals (1,2,3,...)
  4. Lowercase Letters (a,b,c,...z)
  5. Uppercase Letters (A,B,C,...Z)

start attribute:

For <ol> tag, the start attribute is used to specify the starting of numbering.

Basic Example


<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <p>Fruits List.</p>
        <ol type = "a" start = "4">
            <li>Apple</li>
            <li>Mango</li>
            <li>Banana</li>
        </ol>
    </body>
</html>

HTML Images

To represent HTML pages beautifully and to explain content, we use images for that. Our ability to comprehend complex things is aided by visual representations to do that we use images.

It is self closing tag means no need to close it.

<img> tag is used to add an image to the webpage

src is mandatory attribute for <img> tag

src attribute

src attribute defines the path of an image.

There are two ways to specify the URL in the src attribute:

  1. Absolute URL - Links to an external image that is hosted on another website.
    For example:- src = "https://www.tryschools.com/images/img_girl.jpg"
  2. Relative URL - Links to an image that is hosted within the website. Here, the URL does not include the domain name. If the URL begins without a slash, it will be relative to the current page.
    For example:- src = "/images/img_girl.jpg".

alt attribute

The required alt attribute for the <img> tag specifies an alternate text for an image, if the image for some reason cannot be displayed. This can be due to a slow connection, or an error in the src attribute, or if the user uses a screen reader.

height/width attribute

<img> tag uses the height or width attribute to set the height or width of an image. Values for height or width attribute must be in pixel(default) or percentage and value should be placed in double quotes otherwise we can see an error.

Basic Example of Image

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
    <img src = "https://blog.hubspot.com/hubfs/GettyImages-1177482693.jpg" alt="Image Loading" height="100" width="200" >
    <img src = "GettyImages.jpg" alt="Image Loading"> <!-- Broken Image -->
    </body>
</html>

HTML Tables

A table in HTML consists of table cells inside rows and columns.

<table> tag is used to define a table on an HTML page

we can arrange our data in rows and columns of a table.

we can use may tags inside table tag like tr, td, th, etc

tr/td/th tag in Table

Each table row starts with a <tr> and ends with a </tr> tag. tr stands for table row.

Each table cell is defined by a <td> and a </td> tag. td stands for table data.

Sometimes you want your cells to be table header cells. In those cases use the <th> tag instead of the <td> tag. th stands for table header.

    <!DOCTYPE html>
    <html>
        <head>
            <title>Page Title</title>
        </head>
        <body>
            <table>
                <tr>
                    <th>Name</th>
                    <th>Email Id</th>
                    <th>Phone No.</th>
                </tr>
                <tr>
                    <th>John</th>
                    <th>johne22@hotmail.com</th>
                    <th>892909092</th>
                </tr>
                <tr>
                    <th>Alpha</th>
                    <th>alpha48@gmail.com</th>
                    <th>22903999</th>
                </tr>
            </table>
        </body>
    </html>

Rowspan & Colspan attribute

To make a cell span over multiple rows, use the rowspan attribute.

To make a cell span over multiple columns, use the colspan attribute.

    <!DOCTYPE html>
    <html>
        <head>
            <title>Page Title</title>
        </head>
        <body>
            <table>
                <tr>
                    <th colspan = "2" border="1" >Name</th>
                    <th>Phone No.</th>
                </tr>
                <tr>
                    <th>John</th>
                    <th>Marsh</th>
                    <th>892909092</th>
                </tr>
                <tr>
                    <th>Alpha</th>
                    <th>Noman</th>
                    <th>22903999</th>
                </tr>
            </table>
        </body>
    </html>

HTML Links

  • These are specified by <a> tag.
  • HTML LINKS are also known as HYPERLINK.
  • <a> tag is known as an Anchor tag.
  • <a> tag has its corresponding </a> tag.
  • These are used to link one document with another document.

Attributes of Link

  1. href :- Without this tag, we are unable to go to the next page or the next document. This attribute is compulsorily used by the <a> tag. This attribute contains a URL path.
  2. target :- The link tag uses another attribute which is the target attribute. By default, if we open any link then the page will be displayed in the current browser window. But if we wish to change these settings then we use the target attribute. It tells us where we want our linked document to be opened. This target attribute can use any of the values:
    • _blank: Opens linked document in a new window
    • _top: Opens document in the full body of the window
    • _self: Opens document in the same window
    • _parent: Opens linked document in a parent frame

Basic Example

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <a href = "index.html" > Go to Google Page with _self </a><br>
        <a href = "index.html" target = "_blank" > Go to Google Page with _blank </a>
    </body>
</html>

HTML Frames

The HTML <iframe> tag specifies an inline frame.

An inline frame is used to embed another document within the current HTML document.

Tip: It is a good practice to always include a title attribute for the <iframe>. This is used by screen readers to read out what the content of the iframe is.

height/width Attribute

Use the height and width attributes to specify the size of the iframe.

The height and width are specified in pixels by default.

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <iframe src = "index.html" height = "100" width="150" name = "exframe" title = "Home Page" ></iframe>
    </body>
</html>

Target for a Link

An iframe can be used as the target frame for a link.

The target attribute of the link must refer to the name attribute of the iframe:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <iframe src = "index.html" height = "100" width="150" name = "exframe" title = "Home Page"> </iframe>
    <a href = "editor.html" target = "exframe">Go to</a>
</body>
</html>

HTML Forms

Forms are the interactive part of the web page. They are used to take userâ€™s input in different ways. These inputs are taken using various controls. The data collected by these controls can be used for processing, storing etc.

The HTML <form> element is used to create an HTML form for user input

The <form> element is a container for different types of input elements, such as: text fields, checkboxes, radio buttons, submit buttons, etc.

The <input> Element

HTML <input> element is the most used form element.

<input> is self closing tag

An <input> element can be displayed in many ways, depending on the type attribute.

Type Description
text Displays a single-line text input field
email Displays a single-line text input field for email only
password Displays a single-line text input field for passwords (hide characters)
radio Displays a radio button (for selecting one of many choices)
checkbox Displays a checkbox (for selecting zero or more of many choices)
submit Displays a submit button (for submitting the form)
reset Displays a submit button (for reset all the inputs of form)

input tag Attributes

Attribute Description
value defines value to be stored in database
required specifies that an input field must be filled out before submitting the form
height/width specifies height and width of input tag
name a reference when the data is submitted
