What is JS?

  • JS stands for JavaScript
  • JavaScript is the world's most popular programming language. JavaScript is the programming language of the Web. JavaScript is easy to learn. This tutorial will teach you JavaScript from basic to advanced.
  • JavaScript and Java are completely different languages, both in concept and design. JavaScript was invented by Brendan Eich in 1995, and became an ECMA standard in 1997.
  • JavaScript Can Change HTML Content

Features of Javascript

  • Javascript executes the client-side script in the browser.
  • Events are actions. Javascript provides event-handling options.
  • As Javascript is not a compiled language, source code never changes to byte code before running time. Low-end devices can also run Javascript because of its lightweight feature.
  • In Javascript, names, variables, keywords, and functions are case-sensitive.
  • Javascript will run in the same way in all systems with any operating system.

Data Types

JavaScript has 8 Datatypes:

  1. String
  2. Number
  3. Bigint
  4. Boolean
  5. Undefined
  6. Null
  7. Symbol
  8. Object

String Data Type Example

<script>
    let msg = "Hello";
    document.write(msg);
</script>

Number Data Type Example

<script>
    let num1 = 10;
    let num2 = 29.3;
    document.write(num1);
    document.write("<br>");
    document.write(num2);
</script>

Boolean Data Type Example

<script>
    let eval = true;
    document.write(eval);
</script>

Object Data Type Example

<script>
    const person = {firstName:"Mark", lastName:"Harry"};
    document.write(person.firstName+" "+person.lastName);
</script>

Array Object Example

<script>
    const fruit = ["Apple", "Banana", "Litchi", "Strawberry"];
    document.write("First fruit of array is " + fruit[0]);
</script>

Date Object Example

<script>
    const date = new Date("2022-03-25");
    document.write(date.getFullYear());
</script>

Variables

In a programming language, variables are used to store data values.

JavaScript uses the keywords var, let and const to declare variables.

An equal sign is used to assign values to variables.

Keyword Description
var Declares a variable
let Declares a block variable
const Declares a block constant

Basic Example to show difference between var and let

<script>
    let a = 10;
    {
        let a = 70;
        document.write("a (inside block) = " + a+"<br>")
    }
    document.write("a (outside block) = " + a+"<br>");
    var b = 10;
    {
        var b = 70;
        document.write("b (inside block) = " + b +"<br>")
    }
    document.write("b (outside block) = " + b + "<br>");
</script>

JavaScript Functions

A JavaScript function is a block of code designed to perform a particular task.

A JavaScript function is executed when "something" invokes it (calls it).

Basic Syntax

function name(parameter1, parameter2, parameter3) {
    // code to be executed
}

Why Functions?

You can reuse code: Define the code once, and use it many times.

You can use the same code many times with different arguments, to produce different results.

Basic Function Converting temperature Fahrenheit to Celsius 

<script>
    function toCelsius(fah){
        return (5/9) * (fah - 32);
    }
    document.write("28F = " +toCelsius(28) + "C");
</script>

Strings

JavaScript strings are for storing and manipulating text.

A JavaScript string is zero or more characters written inside quotes.

Printing String Message

<script>
    let msg = "Hello";
    document.write(msg);
</script>

String Length

To find the length of a string, use the built-in length property:

<script>
    let msg = "Hello";
    document.write("length of "+msg+" is "+msg.length);
</script>

String Content Replace

The replace() method replaces a specified value with another value in a string.

<script>
    let msg = "Hello, How are you KARAN;
    document.write("Before Replacing: "+ msg + "<br>");
    let msg2 = msg.replace("KARAN","Rohit");
    document.write("After Replacing: "+ msg2);
</script>

Changing Case

We can change the string to uppercase and lowercase

Program for uppercase and lowercase

<script>
    let msg = "Hello, How are you KARAN";
    document.write("Upper Case String: "+ msg.toUpperCase()+"<br>");
    document.write("Lower Case String: "+ msg.toLowerCase());
</script>

Trim String

The trim() method removes whitespace from both sides of a string.

The trimStart() method works like trim(), but removes whitespace only from the start of a string.

The trimEnd() method works like trim(), but removes whitespace only from the end of a string.

<script>
    let msg = "        Hello World     d ";
    document.write("message length: "+msg.length+"<br>");
    document.write("trim length output: "+ msg.trim().length+"<br>");
    document.write("trimStart length output: "+ msg.trimStart().length+"<br>");
    document.write("trimEnd length output: "+ msg.trimEnd().length);
</script>

Arrays

Arrays are object which holds the homogenous type of values

JavaScript arrays are written with square brackets.

Array items are separated by commas.

following code declares (creates) an array called cars, containing three items (car names):

<script>
    const cars = ["Saab", "Volvo", "BMW"];
    document.write(cars);
    document.write("<br>" + "First Element of Array: " + cars[0])
</script>

Array Length

The length property of an array returns the length of an array (the number of array elements).

<script>
    const cars = ["Saab", "Volvo", "BMW"];
    document.write("Length of Array: "+ cars.length);
</script>

Adding Array Element

Push method used to add array element at 

<script>
    const cars = ["Saab", "Volvo", "BMW"];
    document.write(cars+"<br>");
    cars.push("Maruti");
    document.write("Updated Array: "+ cars);
</script>

The unshift method adds a new element to an array (at the beginning), and "unshifts" older elements.

<script>
    const cars = ["Saab", "Volvo", "BMW"];
    document.write(cars+"<br>");
    cars.unshift("Maruti");
    document.write("Updated Array: "+ cars);
</script>

Removing Array Elements

The pop method removes the last element from an array

It returns the removed element

<script>
    const cars = ["Saab", "Volvo", "BMW"];
    document.write(cars+"<br>");
    cars.pop();
    document.write("Updated Array: "+ cars);
</script>

The shift method removes the first array element and "shifts" all other elements to a lower index.

<script>
    const cars = ["Saab", "Volvo", "BMW"];
    document.write(cars+"<br>");
    cars.shift();
    document.write("Updated Array: "+ cars);
</script>
